InteractiveROSETTA - Manual

Abstract: InteractiveROSETTA aims to provide a graphical user interface to the PyRosetta environment to facilitate the use of various Rosetta tools and make these tools more accessible to non-computational researchers. InteractiveROSETTA can be run locally on a personal computer or it can interface with a server in order to offload computationally expensive Rosetta calculations to dedicated machines. Structures can be visualized easily with the PyMOL viewer and a sequence window that makes it easy to select desired segments of protein sequence. The GUI is cross-platform and can run on Windows 7/8, OSX, and Debian/RedHat-based Linux distributions.
